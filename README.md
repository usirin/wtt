WTT - What the Type?
=====

`wtt` is a small library for type checking in JS.

## Version
0.0.1

## Installation

    npm install wtt

## Usage
    var is = require('wtt');

    var fn = function() { return true; }

    is(fn).aFunction(); // returns true
